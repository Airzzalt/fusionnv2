Server Commands & Paid Features

Teams Commands

/team create [name]
Create a new team.
/team invite [player]
Invite a player to your team.
/team leave
Leave your current team.
/team info
View information about your team.
/team chat
Toggle team chat mode.
/team promote [player]
Promote a team member to leader.
Requires Team Leader rank
/team sethome [homename]
Sets the home for the team
Requires Team Leader rank

Lifesteal Commands

/withdraw [amount]
Withdraw hearts into a physical item.
/balance
Check your current heart balance.
/revive [player]
Revive an eliminated player (costs hearts).
/shop
Open the Lifesteal shop to purchase upgrades.
/hub
Teleports user to the Hub

Essential Commands

/spawn
Teleport to the server spawn point.
/tpa [player]
Request to teleport to another player.
/sethome
Set your home location.
/home
Teleport to your set home location.
/msg [player] [message]
Send a private message to a player.
/vote
Get links to vote for the server and receive rewards.
